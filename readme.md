
## 명령어
```
for linux
g++ make-pdu-arr.cpp && cat inp | ./a.out

for windows(cmd)
g++ make-pdu-arr.cpp && type inp | ".\a.exe"
```

## input 파일 형식

```
I X 4.0
M B 2
M D 3
DB1.DB X 0.0
DB7.DB W 6
DB10.DB D 6

메모리 어드레스 바이트.비트

I: 입력 메모리
Q: 출력 메모리
M: 비트 메모리
DB: 데이터 블록

X: bit
B: byte
W: word
D: Dword
```
